Introduction

Cette partie concerne un projet que j'ai mené avec quatre autres étudiants de l'ENSPG (Physique Grenoble) en 2000.

Résumé

L'objectif de ce projet est la réalisation d'un insecte robot à 6 pattes. Nous avons créé les structures de ce dernier dans le but de le faire marcher dans une seule direction. Nous voulions un robot évolutif susceptible d'être modifié pour accomplir de nouvelles tâches. Ce rapport décrit les démarches et les choix du groupe qui ont abouti au fonctionnement du robot.
