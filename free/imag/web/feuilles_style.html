Les Feuilles de style

(ou fichier.css)

Il s'agit en gros de définir de nouveaux paramètres pour toutes les balises HTML.

Le plus simple est de créer un fichier texte en plus des fichiers HTML qui va contenir des définitions de style, de mise en page.
Ainsi on aura plus qu'à faire appel à ce style à chaque fois que l'on en aura besoin.

Pour avoir une documentation de référence, le mieux est de chercher sur Google.

Exemple :

  • Soit le fichier de style : mon_style.css qui contient le code suivant définissant style_cool:

.style_cool {color:green;font-size:10px;}

  • Soit le fichier HTML test.html qui utilise mon_style.css et qui contient le code suivant :

Entre les balises <HEAD> et </HEAD> on appelle la feuille de style
<LINK REL="stylesheet" TYPE="text/css" HREF="mon_style.css">

Dans le corps (<BODY>) on l'utilse avec CLASS
<P>Je suis un paragraphe classique</P>
<P CLASS="style_cool">Je suis un paragraphe personnalisé par le style : style_cool</P>

  • Le résultat est alors

Je suis un paragraphe classique

Je suis un paragraphe personnalisé par le style : style_cool

  • Ce principe est très utile pour définir une charte graphique globale pour un gros site.
