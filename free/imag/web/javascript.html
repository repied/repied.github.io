Javascript

 

Celà consiste à ajouter des (petits) bouts de code au HTML qui seront envoyés au client (Netscape) et exécuté par celui-ci.

Celà sert à donner un peu plus d'interactivité, vérifier que tous les champs d'un formulaire ont bien étés remplis avant de l'envoyer, grossir un image lors d'un clic, changer l'aspect d'un bouton quand la souris passe dessus.

Sa syntaxe ressemble au C et à Java (mais ce n'est pas un langage orienté objet)

Problèmes : Comme les scripts peuvent être interprétés par des clients très différents, il y a souvent des problèmes de compatibilité.
Il faut donc programmer très proprement et tester avec plusieurs navigateurs différents.

Voici un tutoriel bien fait, du même auteur que celui pour HTML.

Exemple :

  • Voici le source:

<SCRIPT LANGUAGE="Javascript">
<!-- ;
function actif(){document.images["bouton1"].src="bouton_enfonce.gif"};
function passif(){document.images["bouton1"].src="bouton_normal.gif"};
// --> </SCRIPT>

<A HREF="INTRO.html" onmouseOver="actif();" onmouseOut="passif();">
<IMG SRC="bouton_normal.gif" NAME="bouton1"></A>

  • Et voilà le résultat

 

Remarque : d'autres language de srcipts existent : un exemple en VBscript
