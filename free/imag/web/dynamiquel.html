Pages dynamiques

 

Il s'agit d'ajouter du code php (gratuit) ou asp (Microsoft) dans la page hébergée par le serveur.

Ce code PHP va générer une page en code HTML qui sera envoyée au client.
Ainsi l'internaute ne pourra jamais voir ce code PHP, il aura disparu de la page quand il recevra celle-ci.

Exemple :

  • Si la page source sur le serveur contient :
    <?php
    echo "    <bold>    Hello World</bold>"
    ?>
  • L'internaute recevera la page dont le source est
    <bold>Hello World</bold>
  • Et qui s'affichera dans son client :
    Hello World

Mais PHP montre vraiment ses possibilités quand il est interfacé avec une base de donnée :

  • Publications de nouvelles
  • Enregistrement dans une base de données venant d'un formulaire
  • Moteur de recherche
  • Caddie électronique
  • Portail avec personnalisation du contenu

Pour pouvoir développer un site dynamique il faut donc installer un seveur web sur son PC

Personal Web Serveur de Microsoft est sur le CD de W98 mais le plus simple est d'utiliser le trio suivant:

  • Apache (serveur du monde unix mais existe aussi sous W9X)
  • PHP3
  • Mysql (gestinnaire de base de donnée gratuit qui s'interface facilement avec PHP)

En effet on peut trouver sur le net un pack tout simple à installer et pré-configuré pour Windows 98 appellé EasyPHP (Disponible sur le site de phpinfo.net ).

Documentations :

  • Tutoriel php3 : http://linux-france.org/article/devl/php3/tut/php3_tut.html
  • Pour avoir une documentation de référence, le mieux est de chercher sur Google.
