Le code HTML

 

C'est un langage interprété par le navigateur (ou client).

Il utilise des balises pour mettre le texte en forme.

Exemples :
<bold> Je suis gras </bold> donne Je suis gras

Le balises peuvent prendre des paramètres :
<a href="intro.html" target="mainFrame">liens vers l'introduction </a>
donne liens vers l'introduction dans la frame (cadre) principal (pas à gauche ou il y a le menu)

Voici un lien vers un tutoriel un peu long mais bien fait pour apprendre le HTML.
Pour avoir une documentation de référence, le mieux est de chercher sur Google.

Pour pouvoir écrire le code source avec les balises, il y plusieurs façons :

  • Notepad : pas pratique
  • des éditeurs spécifiques comme EDITPLUS2 (Version de démonstration disponible sur le site de télécharger.com ) qui mettent les balises en couleurs et permettent d'aller beaucoup plus vite tout en gardant la maitrise du code
  • DreamWeaver qui est graphique mais permet quand même de toucher au code (BON)
  • Frontpage et Netscape qui sont rapides à utiliser mais qui générent un code lourd et incompréhensible : déconseillé
  • La suite Office (option "enregistrer au format html") : Pratique pour traduire un document déjà tapé mais à proscrire en général ( code lourd, redondant, non optimisé, nul...)
  • Immedi@t : Pas besoin de connaissance en html, beau, rapide, code bien généré mais 4000 Fr par site.

Structure d'une page HTML:

<html> Début
<head>
Présentation de la page.
Informations pour les moteurs de recherche.
Titre.
Déclaration des autres langages éventuellement utilisés.
</head>
<body>
Ce qui sera affiché par le navigateur.
</body>
</html> Fin de la page

 

Comme le HTML est limité et donc d'autres langages s'y sont greffés. (javascipt, css, php, asp...)

Remarque :
Le premier fichier à être lu quand le client demande le site à une adresse est index.html
Donc il faut appeller sa page d'accueil index.html.
