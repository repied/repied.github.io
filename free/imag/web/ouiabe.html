Qu'est-ce qu'un site web ?

 

Un ensemble de fichiers textes + des images + des autres documents téléchargeable ou visualisable.

Extentions : .htm ou .html pat défaut mais on peut préciser si il y a un autre langage dedans : .css .php .asp etc...

Ces fichiers sont sur un serveur web qui reçoit des requêtes de clients;

Serveur : logiciel (ex: Apache, PWS) qui tourne sur un gros ordinateur ausi appelé serveur.
Client : Internet Explorer ou Netscape etc... qui tourne sur le PC de l'internaute

Déroulement:

  • le client fait une requête (demande d'afficher une page) selon le protocole http
  • le serveur construit la page à envoyer si c'est un site dynamique, ou va simplement la chercher sur son disque dur si c'est un site simple (statique)
  • le client (Netscape par exemple) recoit cette page au format HTML+javascript
  • il interprète les balises pour faire la mise en page et exécute le code javascript.

Conclusion :

  • la mise en page dépend fondamentalement du client (ie le navigateur) d'ou la difficulté d'avoir un aspect toujours le même pour tous les internautes.
  • Ceci est d'autant plus vrai pour tout ce qui est codé en Javascript.

Pour faciliter et uniformiser la mise en page : utiliser des feuilles de style (.css)

Pour faire des pages plus interactives qu'avec seulement du HTML :

  • mettre dans la page du code javascript qui sera exécuté par le client
  • mettre dans la page du code php ou asp qui sera exécuté par le serveur, avant l'envoi au client (site dynamique)

Remarque :
Le premier fichier à être lu quand le client demande le site à une adresse est index.html
Donc il faut appeller sa page d'accueil index.html.
