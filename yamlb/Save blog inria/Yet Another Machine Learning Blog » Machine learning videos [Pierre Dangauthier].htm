Machine learning videos

March 16, 2007 on 4:12 pm | In Work, research life |

There are several places on the web with intereting ML videos, from research talks or from classes.

O. Bousquet has made a list.
Y. Bulatov points to

  • M. Wainwright class notes with videos.
  • the PASCAL network repository now located on a more general academic website : videolectures.net

Some recent Google TechTalks

  • Hierarchial Dirichlet Processes M. Jordan. “Introduction” to the non parametric Dirichlet process model.
  • Learning Object Appearance Models via Transformed Dirichlet Processes. E. Sudderth. Using Dirichlet for vision.
  • Smoothed-Dirichlet Distribution: A New Generation Building Block R. Nallapati. Improving the Dirichlet-Compound-Multinomial model.
  • Probabilistic Dimensional Reduction with Gaussian Process Latent Variable Model Neil Lawrence. Nice matlab demos.
  • Sparse and large-scale learning with heterogeneous data G. Lanckriet. Dealing with massive and structered data.
  • Using Statistics to Search and Annotate Pictures N. Vasconcelos. Image indexing and retrieval.
  • Learning and Recognizing Visual Object Categories.Vision without feature extraction. Patches are coming back!

It seems to me that infinite models are a hot topic in the bayesian learning community. By infinite, I mean mainly gaussian processes and dirichlet processes.
This last video isn’t related with machine learning but still worth alot watching Gapminder.org
