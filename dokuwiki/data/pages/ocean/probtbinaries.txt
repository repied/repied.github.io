=== Mac OS X ===
==== PyPL binary (no C++ toolkit, just Python runtime)
* tested on Mac OS X Panther 10.3.7 (melon, lunaire, fluo)
* ProBT CVS, as of 12/01/2005
* debug version, no optimisation
* download: http://ocean/probt_macosx.tar.bz2
* instructions:
 # download to your Desktop
 # open a terminal and type:
 % cd ~/Desktop
 % tar xvjf probt_macosx.tar.bz2
 % probt_macosx/pypl
 # => you're inside the Python interpreter, with ProBT loaded
* go to PyPLCookbook if you need advice about using PyPL bindings
