Pour ajouter une nouvelle page:

0 aller sur la page à partir de laquelle on veut ajouter un lien vers la nouvelle page (qui n'existe pas encore)
0 cliquer sur le bouton 'Edit' en bas à gauche
0 ajouter une ligne comme ça:
 * MaNouvellePage (un mot avec plusieurs majuscules sans espace est un nom de page)
0 cliquer sur le bouton Save
0 cliquer sur le lien qui s'est mis pointant vers MaNouvellePage
0 éditer la zone de texte, et cliquer sur Save
Wouala, c'est tout.

Pour le markup hypersimple qu'on peut utiliser pour mettre en forme le texte, voir KwikiFormattingRules. Pour essayer, sans risquer de ne rien casser, il y a le BacASable. (de toute façon des backups réguliers sont faits, et chaque page est soumise à un contrôle de version avec [Subversion http://subversion.tigris.org/], ce qui veut dire que tous les changements sont réversibles, donc pas de danger)
