=== Jam ===
[Jam http://freetype.sourceforge.net/jam/] est un programme qui fait ce que [Make http://www.gnu.org/software/make/] fait, en mieux (voir aussi MakeReplacements sur ce thème). En particulier, il inspecte les dépendances sur les en-têtes tout seul, très rapidement, et on a en général pas besoin de se préocupper des .o. Pour installer:
  apt-get install ftjam
La doc est un peu bizantine, et le principe est assez différent de Make, mais ça vaut le coup. J'ai utilisé ça pendant un moment, mais maintenant j'utiliser Scons: voir UsingScons.
Un exemple :
 # déclaration des sources (.c/.cpp)
 SRC =
        heThing.cpp
        $(MoutonsDIR)/buLaplaceTableManager.cpp
        heThingTester.cpp
        ;

 # chemins des en-têtes
 HDRS =
                $(PLDIR)/pl/include
                $(BURPDIR)/include
                $(MoutonsDIR)
                $(MoutonsDIR)/showDistribGL
                ;

 # options du relieur
 LINKFLAGS =
        -pipe
        -L/scratch/lib
        -L$(HOME)/lib
        -L/usr/X11R6/lib
        -L/usr/lib
        -L$(PLDIR)/lib
        -L$(MoutonsDIR)/showDistribGL
        ;

 # bibliothèques à lier
 LINKLIBS =
        -lpl
        -lShowDistribGL
        -lefence # for debugging (this program is all about testing anyway)
        ;

 C++FLAGS = -g -ggdb -Wall -fPIC -Wstrict-prototypes ;
 CCFLAGS = $(C++FLAGS) ;
 C++ = g++-3.2 ;

 LINK = $(C++) ;

 # exécutable à construire à partir des sources
 Main thingTester : $(SRC) ;
