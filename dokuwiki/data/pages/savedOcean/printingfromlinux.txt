== installing client support for cups ==
[CUPS http://www.cups.org/] is the client-server system used to manage printing. First, make sure the following is installed (these are the names of the debian packages):
* [=cupsys-client]
* [=cupsys-bsd] (to use lpr with cups)
* at least one of [=gtklp] and [=qtcups]

Then make sure the following line is in your /etc/cups/client.conf:
  ServerName print-serv.inrialpes.fr
You should comment out any other line starting with !ServerName.

Then make sure you set your PRINTER environment variable to something useful. For instance, I have in my .cshrc:
  setenv PRINTER le212a
bash/sh/zsh version:
  export PRINTER=le212a
[=le212a] is the black & white printer on the second floor of corridor E. There's a color printer on the first floor, [=le110b] IIRC.

== printing ==
Several possibilities:
* just set the printing options for all printing jobs in the future (this lets you edit the number of pages by sheet, the default printer, the binding edge, landscape/portrait mode...) :
  qtcups # or gtklp
* simply print from the command line using the current options:
  lpr toto.txt
  lpr toto.ps
  lpr toto.pdf
* print using qtcups or gtklp, to get a nice printing options dialog box before anything gets sent:
  gtklp toto.pdf
* consult the printing queue for the current printer:
  lpq
* same thing, graphical interface:
  gtklpq
