[SCons http://www.scons.org/] is a powerful and friendly replacement for Make (for me, the best of all MakeReplacements -- RonanLeHy).

The complete !SConstruct file that I use to build my moutons follows. It's mostly self-explanatory, I think. Just a few notes:
* to compile, just type
  scons
* this is actually a Python script
* you don't have to specify your .h files, or dependencies, it's all resolved automatically
* scons won't recompile twice your file if it intervenes in two different targets (which is great)
* scons will recompile everything correctly if you change the compilation options (like the -I or -L flags for instance), or if you change the compiler
* scons can do many wonderful things -- they say it can replace the autotools also
* please add any trick you have in scons to this page -- an example is often enough
* the most interesting beginner's documentation is probably the [user guide http://www.scons.org/doc/HTML/scons-user.html]

!SConscript file in src directory in the root project directory:
 # -*- Mode: python -*-
 # a scons build script for the EnvOccGrid library
 # Manuel Yguel, june 2004

 # customise these vars only ##########
 cxx = 'g++-3.3'

 projectdir = '..'

 pldir = '/local_home/manu/code/PL'

 debugdir = '/home/andrinople/yguel/code/include'


 cgnuplotdir = '/local_home/manu/code/CGnuplot'


 cycabdevptincludedir = '/local_home/manu/code/robot/CycabDevpt/'
 ray_modeldir = '/local_home/manu/code/robot/TelemSensMod/'
 ######################################

 # to access the outside environment
 import os
 def en(x):
     return os.environ[x]


 # the sources
 src = Split("""
 EnvOccGrid.cpp
 """)


 # the include path
 cpppath = Split("""
 ${PROJECTDIR}/include
 ${PLDIR}/source/pl/include
 ${DEBUGDIR}


 # ${ENVMAPDIR}
 # ${DEFINESDIR}

 ${CYCABDEVPTINCLUDEDIR}/include
 ${RAY_MODELDIR}/include

 """)

 linkflags = """
 -pipe
 """

 libpath = Split("""
 ${PLDIR}/source/lib
 ${CGNUPLOTDIR}
 ${CYCABDEVPTINCLUDEDIR}/lib
 ${RAY_MODELDIR}/lib
 ${PROJECTDIR}/lib
 """)

 libs = Split("""
 CGpl

 EnvOccGrid
 CLM
 VirtualSick
 TelemSensMod
 pl
 V2

 """)



 #Get the user value
 debug = ARGUMENTS.get('debug', 1)
 print '===== Debug mode : ' + repr(debug)

 #Debug directives
 if '0' == debug:
     cxxflags = '-Wall -W -O3 -Wstrict-prototypes '
 else:
     cxxflags = '-g -Wall -W -Wstrict-prototypes '

 #A build environment
 env = Environment(
     PROJECTDIR = projectdir,
     PLDIR = pldir,
     DEBUGDIR = debugdir,
     CGNUPLOTDIR =  cgnuplotdir,

     RAY_MODELDIR = ray_modeldir,

     CYCABDEVPTINCLUDEDIR = cycabdevptincludedir,

     HOME = en('HOME'),

     CPPPATH = cpppath,
     CXX = cxx,
     CXXFLAGS = cxxflags,
     LIBPATH = libpath,
     LIBS = libs
     )

 # A library target
 lib = env.Library( target = 'EnvOccGrid', source = src )
 env.Install("${PROJECTDIR}/lib", lib)
 env.Alias('lib', "${PROJECTDIR}/lib")
