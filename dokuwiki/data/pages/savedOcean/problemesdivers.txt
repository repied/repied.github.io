=== problèmes de TLS (thread local store) ===
Si des applis OpenGL (particulièrement utilisant SDL, comme frozen-bubble (apt-get install frozen-bubble))  refusent de tourner avec le message suivant :
  libGL.so.1: cannot handle TLS data
Lancez-les comme ça :
  LD_PRELOAD=/usr/lib/tls/libGL.so frozen-bubble
Ou ajoutez dans vos fichiers de démarrage (.cshrc, .environment, ou ce que vous utilisez) :
  setenv LD_PRELOAD /usr/lib/tls/libGL.so # syntaxe csh/tcsh
Ou si vous utilisez bash :
  export LD_PRELOAD=/usr/lib/tls/libGL.so # syntaxe sh/bash/zsh
Et pour recharger son .cshrc quand on l'a modifié :
  source ~/.cshrc
