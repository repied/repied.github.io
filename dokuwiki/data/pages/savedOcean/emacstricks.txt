=== getting help ===
* M-x describe-bindings : lists all currently available key bindings
* M-x describe-mode : describes the current active modes
* [XEmacs reference card http://www.digilife.be/quickreferences/QRC/XEmacs%20Reference%20Card.pdf]

=== M-C-cokebottles ===
* fill-paragraph: C-c f
* shell-command (new buffer): M-!
* shell-command (insert result in current buffer): C-u M-!
* shell-command-on-region: C-u M-| (filtre la region selectionnee, sur place)
* query-replace: M-%
* enregistrer une macro: C-x ( <ici faire des trucs à enregistrer> C-x )
* call-last-kbd-macro: C-x e
* describe-key: C-h k (ou : ça fait quoi, cette combinaison de touches ?)
* comment-region: C-c C-c (en mode C/C++ en tous cas, pas LaTeX)
* uncomment-region: C-u C-c C-c
* undo: C-_
* goto-line: M-g (par défault seulement dans xemacs)
* c-set-style: C-c . (en mode C/C++, permet de changer le style d'indentation, mon préféré est stroustrup)

=== fonctions utiles, sur lesquelles il est intéressant d'ajouter des raccourcis ===
* replace-string
* replace-regexp
* compile (a des bindings préexistants différents selon les modes, LaTeX et C++ par exemple)
* indent-region

=== entrer des accents avec un clavier qwerty ===
 M-x set-input-method french-postfix
(permet aussi d'entrer du Russe, du Chinois, etc. : M-x set-input-method TAB pour voir les methodes d
entree disponibles)

=== ajouter un raccourci pour une fonction existante ===
Dans son .emacs, ajouter:
  (global-set-key "\M-g" 'goto-line)
pour associer M-g à goto-line. Pour que ça prenne effet tout de suite, se placer à la fin de cette ligne, et faire C-x C-e (eval-last-sexp). C'est pareil pour tous les trucs qu'on ajoute dans le .emacs.

=== effacer un raccourci clavier ===
C-z sous X est souvent nuisible. Pour le désactiver, ajouter dans son .emacs :
  (global-unset-key "\C-z")

=== créer une nouvelle fonction à partir d'une macro ===
* enregistrer une macro avec C-x( ... C-x )
* M-x name-last-kbd-macro <Return> ma_nouvelle_fonction
* aller dans son .emacs et faire M-x insert-kbd-macro <Return> ma_nouvelle_fonction
* ensuite on peut associer ça à C-z C-a avec (dans le .emacs toujours) :
  (global-set-key "\C-z\C-a" 'ma_nouvelle_fonction)

=== other tricks ===
* désactiver le mode outline-minor-mode, par Christophe:
Si comme moi vous ne supportez pas les libertes que prend emacs en mode latex (par exemple afficher la lettre grecque quand vous tapez \rho, ou "mettre en page" les indices et exposants), il faut desactiver le "outline-minor-mode", mis par default dans emacs21.

Il faut ajouter dans le .emacs:

 (add-hook
  'LaTeX-mode-hook
  (function
   (lambda ()
     (outline-minor-mode f)  ;; desactivate the *wonderful* Outline mode
   )
  )
 )

Ca doit aussi fonctionner pour xemacs.

* Appliquer une fonction a la region active, ou a la ligne courante si rien n'est selectionne (ne marche probablement tel quel que dans XEmacs).
 (defun region-or-line (f)
  "apply f on the region, if there is one, or the current line"
  (interactive)
  (if (region-active-p)
      (funcall f (region-beginning) (region-end))
    (save-excursion
      (beginning-of-line ())
      (let ((start (point)))
        (end-of-line ())
        (funcall f start (point))))
    ))
 (defun comment-region-or-current-line
  ()
  "comment region of current line (if nothing is selected)"
  (interactive)
  (region-or-line 'comment-region))
 (global-set-key [(control q) (control c)] 'comment-region-or-current-line)
